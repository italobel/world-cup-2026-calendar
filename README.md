# FIFA World Cup 2026 — Calendar (.ics)

All 104 matches of the 2026 FIFA World Cup as a single `.ics` calendar file. Drop it into your calendar once and every match shows up in your schedule — kickoff times, venues, group stage, knockouts, and the final.

The tournament runs **June 11 – July 19, 2026** across the United States, Canada, and Mexico.

## Install in your calendar

**Apple Calendar (Mac / iPhone / iPad)**
1. Download the `.ics` file.
2. Double-click it on Mac, or tap it on iPhone/iPad.
3. Apple Calendar will ask which calendar to add the events to. Pick a new calendar so you can hide it after the tournament if you want.

**Google Calendar**
1. Download the `.ics` file to your computer.
2. Open [Google Calendar](https://calendar.google.com) in a browser.
3. Click the gear (Settings) → **Import & export** → **Import**.
4. Select the file, pick the calendar to add it to, and click **Import**.

**Outlook**
1. Download the `.ics` file.
2. Open Outlook → **File** → **Open & Export** → **Import/Export** → **Import an iCalendar (.ics) or vCalendar file (.vcs)**.
3. Select the file and choose **Import** to add events to your existing calendar.

## What's in the file

- All 104 matches, numbered in order (`WC2026 #1`, `WC2026 #2`, …).
- Group letter for group-stage games (Group A through Group L).
- Kickoff times in **US Eastern Time**, with the file's timezone block set so your calendar app auto-converts to whatever timezone you live in.
- Venue and city for every match (Estadio Azteca, MetLife Stadium, BMO Field, AT&T Stadium, etc.).
- A short description on each event for quick reference.

## Notes

- **Unofficial / fan-made.** This is my own calendar file. It is not affiliated with FIFA. If you spot a mistake in dates, times, or venues, open an issue and I will fix it.
- **Times in Eastern.** The file declares its timezone as `America/New_York`. Modern calendar apps will convert this to your local timezone automatically. If you see Eastern times on your calendar, your app is ignoring the timezone block — try a different import method.
- **One-time import.** If FIFA updates the schedule (rare, but possible for weather or operational reasons), you will need to download a refreshed file. Watch this repo for updates.
